# Blockchain Consensus Percolation Framework

This repository contains the Monte-Carlo simulation scripts, parameter configuration files, processed numerical datasets, and plotting routines associated with the study:

## Finite-Size Phase Transitions in Blockchain Consensus Networks Under Stochastic Connectivity Degradation

The framework investigates quorum-aware percolation, consensus-feasibility thresholds, finite-size scaling, susceptibility behaviour, and adversarial connectivity degradation in stochastic blockchain communication overlays.

The study models blockchain communication systems as bond-percolated stochastic graphs and analyses the emergence of quorum-capable giant connected components under connectivity degradation, targeted attacks, and finite-size stochastic fluctuations.
